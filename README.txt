# Customer Data Masking using IICS

## Overview
This project demonstrates an ETL workflow using Informatica IICS to mask sensitive customer data using substitution masking.

## Features
- Data Masking Transformation
- Substitution Masking
- GDPR-style data protection
- Oracle simulation using CSV

## Tools Used
- Informatica IICS
- Oracle
- SQL
